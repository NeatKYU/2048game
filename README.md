# 2048game
리액트로 만든 2048게임입니다.
리액트로 개발을 진행한지 오래되어서 까먹지 않으려고 진행한 프로젝트입니다.
간단한 방향키로 조작할 수 있으며, 점수가 표시되며, 모드를 선택할 수 있고 (4x4, 5x5) 약간 쓸데없지만 다크모드를 지원합니다.

## Getting Started
```
git clone https://github.com/NeatKYU/2048game.git
cd 2048game
npm install
npm run dev
```
